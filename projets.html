site web musée de l'infini

Création en groupe d'un musée en ligne sur le thème de l'infini.

×

site web musée de l'infini

HTML

CSS

JS

PHP

Dans le cadre du projet SAE (Projet de semestre) du 3e semestre de ma formation sur la création d'un musée basé sur l'infini, j'ai du créer le dite web pour notre groupe, en suivant une identité visuelle, aussi crée par notre groupe.
(Voir projet "Musée de l'infini")

Ce site web est côté en HTML et CSS, il possède une fonctionnalité de traduction via une checkbox sous forme de slider, codé en JavaScript. Toutes les informations des expositions du musée (noms, oeuvres, vidéos, descriptions, etc) se trouvent dans une base de donnée locale (PhpMyAdmin). J'ai donc aussi codé toute la partie PHP avec une architecture MVC.

Voir le site sans PHP
